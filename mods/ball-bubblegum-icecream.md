# Bubblegum Icecream Ball

A simple re-texture of the basic ball into a bubblegum icecream one.

## Details

- **Author:** Tandashi
- **Version:** 1.0.0
- **Tags:** Texture Swap

## Installation

Install this mod through the Pirate Radio Launcher. It will replace the default basic ball texture with a bubblegum icecream themed one.

## What it changes

- `basic_ball_albedo` - Replaces the ball's color/albedo texture
- `basic_ball_normals` - Replaces the ball's normal map for proper lighting
